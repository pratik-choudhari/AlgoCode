Problem Description : 

Given a string containing of ‘0’, ‘1’ and ‘?’ wildcard characters, 
generate all binary strings that can be formed by replacing each wildcard character by ‘0’ or ‘1’.

Sample I/O:
Input str = "1??0?101"
Output:
10000101
10001101
10100101
10101101
11000101
11001101
11100101
11101101
