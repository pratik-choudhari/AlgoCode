# Problem Statement
Given an expression as string comprising of opening and closing characters
of parentheses - (), curly braces - {} and square brackets - [], we need to
check whether symbols are balanced or not.

##Example
Input: exp = “[()]{}{()()}”
Output: Balanced

Input: exp = “[(])”
Output: Not Balanced
